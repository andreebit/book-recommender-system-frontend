# Instrucciones #
## Clonar el repositorio ##
Ejecutar el comando:  
```
git clone https://github.com/andreebit/book-recommender-system-frontend.git
```
___
## Descargar las dependecias ##
En la raiz del proyecto, ejecutar el comando:  
```
npm install
```
### Iniciar el servidor
En la raiz del proyecto, ejecutar el comando:  
```
npm run serve
```
## Probar el sistema ##
Acceder a la url:
```
http://localhost:8080
```
